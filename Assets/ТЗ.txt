Разработать модуль для обеспечения выполнения команд и сценариев в запушенном софте

Элементы:
- Окно консоли
- Интерфейс команды
- Атрибут сценария

Поведение:
- При нажатии на ` открывается окно консоли и прокидывается события открытия
- Вводится название команды или сценария и необходимые аргуменды
- В нижней части окна показывается подсказки, наподобии автодополнения команды, или аргументов метода
- Если команды запускается из MonoBehaviour, то это должно быть обозначино
- В консоль могут выводиться логи

Органы управления:
- `
- Enter
- Up key
- Down key
- Return

Замечания:
- Command для статических функций 
- Scenario для методов классов, наследующих от MonoBehaviour
